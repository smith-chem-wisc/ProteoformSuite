Detailed Description

On-disk mzML implementation of FullSwathFileConsumer.

Writes all spectra immediately to disk to an mzML file location using the PlainMSDataWritingConsumer. Internally, it handles n+1 (n SWATH + 1 MS1 map) objects of MSDataCachedConsumer which can consume the spectra and write them to disk immediately.

Warning: no swathmaps (MS1 nor MS2) will be available when calling retrieveSwathMaps() for downstream use.
