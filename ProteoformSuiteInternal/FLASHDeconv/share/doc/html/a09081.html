Detailed Description

WindowMower augments the highest peaks in a sliding or jumping window.

Parameters of this class are:

NameTypeDefaultRestrictionsDescription
windowsize float50.0  The size of the sliding window along the m/z axis.
peakcount int2  The number of peaks that should be kept.
movetype stringslide slide, jumpWhether sliding window (one peak steps) or jumping window (window size steps) should be used.
