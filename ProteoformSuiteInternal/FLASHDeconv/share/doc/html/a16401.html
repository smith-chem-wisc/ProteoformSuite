OpenSwathRewriteToFeatureXML

Combines featureXML and mProphet tsv to FDR filtered featureXML.

The command line parameters of this tool are: 


OpenSwathRewriteToFeatureXML -- Combines featureXML and mProphet tsv to FDR filtered featureXML.

Usage:

  OpenSwathRewriteToFeatureXML <options>



Options (mandatory options marked with '*'):

  -csv <file>           MProphet tsv output file: "all_peakgroups.xls" (valid formats: 'csv')

  -featureXML <file>*   Input featureXML file (valid formats: 'featureXML')

  -out <file>*          Output featureXML file (valid formats: 'featureXML')

  -FDR_cutoff <double>  FDR cutoff (e.g. to remove all features with a an m_score above 0.05 use 0.05 here) 

                        (default: '-1.0')
