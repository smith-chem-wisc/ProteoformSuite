Requirements
	- JRE 1.6 or greater
	- Main memory 2GB or greater (recommended 4GB)

Installation
	- Place luciphor2.jar in any folder

Usage Information
	- Type 'java -jar luciphor2.jar' for command line arguments
	- For detailed documentation, visit http://luciphor2.sourceforge.net/

Contact Information
	- If you are having problems with the software, please submit a post to the support forum or email at: luciphor-support@googlegroups.com

Publications
  - LuciPHOr: algorithm for phosphorylation site localization with false localization rate estimation using modified target-decoy approach (doi: 10.1074/mcp.M113.028928)
  - LuciPHOr2: site localization of generic post-translational modifications from tandem mass spectrometry data. (doi: 10.1093/bioinformatics/btu788)
